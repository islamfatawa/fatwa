السؤال:

هل يجوز جماع الزوجة في الحمام؟ وهل هناك أماكن لا يجوزالجماع فيها؟

الجواب:

الحمد لله، والصلاة والسلام على رسول الله، وعلى آله وصحبه، أما بعد: فيجوز للزوج جماع زوجته في الحمام، وإن كان الأولى تجنب ذلك لمنافاته للآداب، ولكونه ليس محلا للإتيان بالدعاء المأثور قبل الجماع. وراجع في ذلك الفتوى رقم: 51742 . والجماع إنما يحرم في المسجد لوجوب تنزيهه عن ذلك، ففي الموسوعة الفقهية: وكذلك يحرم فيه الجماع لقوله تعالى: { ولا تباشروهن وأنتم عاكفون في المساجد } انتهى وقال الكاساني في بدائع الصنائع: وكذا الوطء في المسجد حرام، قال الله عز وجل { ولا تباشروهن وأنتم عاكفون في المساجد } انتهى ويكره الجماع بحضور الناس. جاء في مطالب أولي النهى ممزوجا بغاية المنتهى للرحيباني الحنبلي : ( و ) كره ( مباشرتها ) أي : وطؤها ( بحضرة الناس إن كان ) أي الزوجان ( مستور العورة ) لأنه دناءة، ( وإلا ) يكونا مستوري العورة ( حرم ) عليه ذلك مع رؤية العورة؛ لحديث : { احفظ عورتك } انتهى والله أعلم.
